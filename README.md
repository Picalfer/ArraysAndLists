# ArraysAndLists

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Тестовое приложение для изучения работы с массивами и списками в Kotlin.

## О проекте

Приложение демонстрирует различные способы создания и работы с массивами и списками в Kotlin. Изучаются основные операции и методы для работы с коллекциями данных.

**Основной функционал:**
- Создание массивов разных типов
- Работа со списками List и ArrayList
- Операции с массивами (сортировка, реверс)
- Добавление и удаление элементов
- Преобразование между массивами и списками

## Технологии

* Kotlin
* Arrays
* Lists
* Collections
* Logging
